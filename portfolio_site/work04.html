핵심 코드 코드 전체보기

1. 리뉴얼된 메인 페이지

기존사이트(첫번째 이미지)의 과도한 정보제공으로 정보전달력이 다소 낮아, 사용자가 중요한 정보에 바로 접근가능하도록 리뉴얼 사이트(두번째 이미지)를 기획 및 구현했습니다.

2. 회원가입, 아이디 찾기의 유효성 검사

1.  input 태그 내에 required를 추가했으며, input내에 최소 1글자 이상 기입된 것이 확인될 때 alert을 띄우도록 설계했습니다.

2.  모든 checkbox의 속성값이 checked 된 경우 회원가입으로 이동하는 alert창이 실행되도록 구현했습니다.

3. '직접입력' 선택시 input 활성화

attr메서드를 이용, select된 경우에 한해 창을 활성화했으며, 다른 option이 선택된 경우에는 비활성화되도록 구현했습니다.
