핵심 코드 코드 전체보기

1. photoshop을 이용한 배너디자인

포트폴리오 사이트에서 가장 중요한 본인의 작업물을 효과적으로 제시하기 위해 device별로 모든 배너이미지를 직접 제작했습니다.

2. UI적 요소로 표현한 슬라이드 이동 효과

다음 슬라이드까지의 로딩 시간에 대해 직관적으로 파악할 수 있도록 animation 효과를 적용하여 최적화된 UI를 제시합니다.

3. each 메서드 이용한 progress bar 구현

1.  progress bar를 구현하기 위해 html내에 data값을 부여하였으며, each 메서드를 활용하여 해당 데이터 값 만큼 위치가 이동할 수 있도록 설계하였습니다.

2.  data값만 변경하면 바로 수정이 가능하여 누구나 쉽게 값을 수정할 수 있습니다.

3.  skill 항목이 보이는 곳에서 해당 함수가 한 번만 실현되도록 scroll함수를 이용합니다.

4. padding-bottom으로 정원 만들기

가상선택자 'after'과 'padding-bottom'을 이용하여 반응형 페이지에서도 항상 가로와 세로값이 동일한 원형을 만들었습니다.
